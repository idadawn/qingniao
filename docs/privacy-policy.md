# 青鸟（Qingniao）隐私政策

**最后更新：2026 年 3 月 25 日**

## 概述

青鸟（以下简称"本扩展"）是一款将浏览器书签与飞书多维表格进行双向同步的 Chrome 扩展。本政策说明本扩展如何处理用户数据。

## 数据收集与使用

**本扩展不收集、不上传、不出售任何用户个人数据。**

本扩展在运行过程中会访问以下信息，所有数据**仅存储在您的本地设备**（`chrome.storage.local`）：

| 数据类型 | 用途 | 存储位置 |
|----------|------|----------|
| 飞书 App ID 和 App Secret | 与飞书 API 进行身份认证 | 本地设备 |
| 飞书 OAuth 访问令牌（access_token / refresh_token） | 维持与飞书账号的连接 | 本地设备 |
| 飞书多维表格 URL | 定位同步目标表格 | 本地设备 |
| 上次同步时间、同步状态 | 在 Popup 界面展示同步信息 | 本地设备 |

## 书签数据

本扩展会读取您的浏览器书签，并将其同步至您**自己的**飞书多维表格。书签数据直接从您的浏览器发送至飞书开放平台（`open.feishu.cn`），不经过任何第三方服务器中转。

## 网络请求

本扩展仅向以下域名发起网络请求：

- `https://open.feishu.cn`：飞书官方开放平台 API，用于认证和读写多维表格数据。

本扩展不包含任何追踪代码、广告 SDK 或数据分析服务。

## 权限说明

| 权限 | 用途 |
|------|------|
| `bookmarks` | 读取书签以同步至飞书；监听书签变动以实时推送 |
| `storage` | 在本地保存飞书配置和令牌 |
| `identity` | 发起飞书 OAuth 授权流程 |
| `notifications` | 同步完成或失败时显示系统通知 |

## 数据安全

您的飞书凭证仅存储在本地设备，不会被传输至除飞书官方服务器之外的任何地方。您可以随时通过清除扩展数据（Chrome 设置 → 扩展程序 → 青鸟 → 清除数据）删除所有本地存储的信息。

## 联系方式

如对本隐私政策有任何疑问，请通过 Chrome Web Store 商店页面的支持渠道联系开发者。
